## Work

- [systems-engineering-traceability-lab](https://github.com/akifitu/systems-engineering-traceability-lab): requirements, interfaces, risks, and verification linkage in one audit-ready repo.
- [verification-readiness-dashboard](https://github.com/akifitu/verification-readiness-dashboard): static readiness dashboard for closure state, evidence, blockers, and overdue work.
- [embedded-systems-collection](https://github.com/akifitu/embedded-systems-collection): host-buildable C firmware portfolio covering control, diagnostics, safety, and protection logic.
- [embedded-rtos-sim](https://github.com/akifitu/embedded-rtos-sim): preemptive scheduling, semaphores, queues, and defense-style real-time scenarios.
- [guidance-algorithm](https://github.com/akifitu/guidance-algorithm): PN, APN, CLOS, and pursuit-law comparison through 2D engagement simulation.
- [target-tracking-system](https://github.com/akifitu/target-tracking-system): multi-target tracking with Kalman filters, gating, and Hungarian association.
