## About

> Portfolio-grade repositories focused on **traceability**, **verification readiness**, **host-buildable firmware**, **RTOS behavior**, **guidance**, and **tracking**.

- Systems engineering: requirements, interfaces, risks, verification linkage
- Embedded software: portable C, defensive programming, deterministic behavior
- Simulation: guidance algorithms, Kalman filtering, radar tracking, comm-link budgets

**TR:** Bu profilin ana odağı sistem mühendisliği, gömülü yazılım ve teknik simülasyon çalışmaları.
**EN:** Most defense-style scenarios are fictional and used to demonstrate engineering workflow, architecture reasoning, and technical communication quality.
